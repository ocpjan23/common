# ocpjan23 commons

A component library with React, Typescript and Storybook.


## Getting Started

### System Requirements

- git v2.39 or greater
- nodejs `20.11`
- npm `10.2.14`

All of these must be available in your `PATH`. To verify things are set up
properly, you can run this:

```shell
git --version
node --version
npm --version
```


###  Run Locally

Install dependencies

```bash
  npm install
```

To build library

```bash
  npm run build
```

To start storybook

```bash
  npm run storybook
```

To test the library

```bash
  npm run test 
```
